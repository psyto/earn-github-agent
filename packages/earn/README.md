# Earn Frontend Service

Frontend service (Next.js/React) for the Superteam Earn Sponsor Dashboard.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy `.env.example` to `.env` and configure:
```bash
cp .env.example .env
```

3. Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`.

## Components

- **GithubReviewButton**: Button to trigger GitHub reviews
- **ReviewResults**: Component to display review scores, notes, and labels
